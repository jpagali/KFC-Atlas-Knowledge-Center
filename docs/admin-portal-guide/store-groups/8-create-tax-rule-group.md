# Create Tax Rule Group

## What this guide covers

Creates a tax rule group that bundles multiple tax rules together for application across a store group.

## Steps

**Step 1:** Start by going to the Store Groups screen by clicking here.

**Step 2:** Once you find the store group you are looking for, click on the stacked dots to open the option window.

**Step 3:** Click taxes

**Step 4:** Click on tax rule groups tab

**Step 5:** Click this button to create a new tax rule group

**Step 6:** Create the name, display name, description and then hit create tax group to make the tax rule group

## Notes

:::note
You can edit, copy, and delete tax rule groups.
:::
