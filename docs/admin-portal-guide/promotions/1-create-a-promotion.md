# Create a Promotion

## What this guide covers

Builds a new promotional rule in Atlas — defining the discount type, conditions, validity period, and applicable stores or store groups — so it can be surfaced to customers through digital ordering channels.

## Steps

**Step 1:** Start by going to the Promotions screen by clicking here.

**Step 2:** To create a new promotion, click here.

**Step 3:** Choose a promotion flow based on the type of promotion you want to create.   If the promotions isn’t a new price, buy 1 get 1, or reduced price click custom promotion.

**Step 4:** (Add Requirement)  Based on what flow select certain requirements will be recommended.  (click add to add them)

**Step 4:** (Add Requirement)  Select requirement type then click add requirement ( you can also utilize the recommended requirements below)

**Step 5:** Add effect and fill out effect information.

**Step 6:** Review your information next and click create to create the promotion.

## Additional information

- Promotions - Create a Promotion
- Here you can search for promotions and redemption codes and filter by automatic or archived
- This is the Promotions screen where you  will see a list of all the promotions you have created, create new promotions, search for any you have created, edit and copy, add extra info in the Meta link and  assign them to Store Groups.  Promotions can only assigned to a Store Group and not a singular store.
- Fill out promotion name, display name (the name that will be displayed on your chosen channels), and the description.
