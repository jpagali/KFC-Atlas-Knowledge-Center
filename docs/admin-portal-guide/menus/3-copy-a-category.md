# Copy a Category

## What this guide covers

Duplicates a category to speed up menu structuring.

## Steps

**Step 1:** Start by going to the Menu screen by clicking here.

**Step 2:** Click on the categories folder

**Step 3:** Click this  button in the same row the category you’re looking for is in and then hit Copy

**Step 4:** Create a category code and rewrite the category name if needed. The information you want to and hit save afterwards. (By default information from the original category like images and tags will be copied)
