# Delete a Category

## What this guide covers

Removes a category from a menu when the grouping is no longer needed.

## Steps

**Step 1:** Start by going to the Menu screen by clicking here.

**Step 2:** Click on the categories folder

**Step 3:** Click this  button in the same row the category you’re looking for is in and then hit Delete

**Step 4:** Click delete button to delete the category

## Notes

:::note
Deleting a category will remove from all menus
:::
