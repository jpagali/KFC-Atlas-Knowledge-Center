# Assign a Menu

## What this guide covers

Links a menu to a specific store and channel so the correct catalogue is served to customers ordering through that channel.

## Steps

**Step 1:** Start by going to the Menu screen by clicking here.

**Step 2:** Click this button in the same row the menu you’re looking for is in and then hit Assign

**Step 3:** Select stores to assign menu to.

**Step 4:** Select what channels that menu will be tied to

**Step 5:** Click assign
