# Delete a Patch

## What this guide covers

Permanently removes a patch from the system.

## Steps

**Step 1:** Start by going to the Menu screen by clicking here.

**Step 2:** Click on the patches tab

**Step 3:** Click this button in the same row the patch you’re looking for is in and then hit Delete

**Step 4:** Click this button to delete the patch.

## Notes

:::note
Deleting this patch will remove it from all related menus.
:::
