# Create a Patch

## What this guide covers

Creates a menu patch — a targeted override that modifies specific items in a menu without replacing the whole menu, used for localised pricing or availability changes.

## Steps

**Step 1:** Start by going to the Menu screen by clicking here.

**Step 2:** Click on the patches tab

**Step 3:** Click create new paatch

**Step 4:** Create a name for the patch

**Step 5:** Select a operation for the patch

**Step 6:** Click add operation after selecting a patch operation

**Step 7:** Select the items you need to add related to your patch operation, after you select a operation press add operation.

**Step 8:** Click create to create the patch

## Notes

:::note
Here you can see your patch added to the table.
:::

:::note
You can edit, reorder, copy, and delete patches you’ve added to this list.
:::

:::note
You can continue to add more patches by clicking the “select operation dropdown”
:::
