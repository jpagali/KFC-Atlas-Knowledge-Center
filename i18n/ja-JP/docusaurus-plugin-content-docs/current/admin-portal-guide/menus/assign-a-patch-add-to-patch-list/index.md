# a パッチ (Add to パッチ List)を割り当てる

## このガイドで扱う内容

このガイドでは、Byte Commerce Admin Portal でa パッチ (Add to パッチ List)を割り当てる手順を説明します。

## 手順

**ステップ 1:** まず、こちらをクリックして Menu 画面に移動します。
**ステップ 2:** on the patches tab をクリックします。

![Step 2](./img/step-2.png)

**ステップ 3:** create new paatch をクリックします。

![Step 3](./img/step-3.png)

**ステップ 4:** add patch to list を選択します。

![Step 4](./img/step-4.png)

**ステップ 5:** patch を選択します。

![Step 5](./img/step-5.png)

**ステップ 6:** Choose patch position

![Step 6](./img/step-6.png)

**ステップ 7:** a channel を選択します。

**ステップ 8:** stores を選択します。

**ステップ 9:** save をクリックします。

![Step 7](./img/step-7.png)

## 注意事項

:::note
Review stores to assigned patches/position
:::
